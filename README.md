# score-tracker-web

A web application for registering scores in basketball pickup games. The app will feature simple one-click buttons to add 1, 2, or 3 points to either of the two teams' scores. It will display the current score prominently for easy viewing during the game.

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/score-tracker-web.git
cd score-tracker-web
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
